# Détails du Projet : Citations

### Contexte et Objectif
Petit outil qui permet l'affichage aléatoire de citations au clic de souris sur un bouton. La génération se fait d'après une liste que j'ai définie dans un fichier JavaScript.

### Technologies mobilisées
* **Langage** : JavaScript (Manipulation du DOM)
* **Structure** : Tableaux de données JS
