# Détails du Projet : Klivio

### Contexte et Objectif
Intégration web réalisée au départ d'une maquette fournie. Le but était de reproduire fidèlement le design responsive d'une plateforme de formation.

### Technologies mobilisées
* **HTML/CSS** : Pour la structure et le positionnement.
* **Tailwind CSS** : Utilisé pour accélérer le stylisage et garantir le responsive.
