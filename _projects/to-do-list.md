# Détails du Projet : To-do List

### Contexte et Objectif
Création d'un formulaire permettant le stockage d'une liste de tâches. Grâce à l'utilisation du localStorage, la liste est automatiquement rechargée quand on ouvre la page.

### Technologies mobilisées
* **Langage** : JavaScript
* **Stockage** : localStorage pour que la liste soit rechargée automatiquement à l'ouverture de la page.
